June, 30

media attribute describes the device for which the style design should be applied.

rel attribute describes the external document's link type.

target attribute of a tag specifies where to open the linked document. New page, current page, etc.

base tag describes base link address for every document link.

meta tag defines document metainformation.

Open Graph is fucking facebook protocol which allows to create beautiful links for social networks.
